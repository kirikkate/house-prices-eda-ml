# House Prices: EDA & Regression Models

## Project description
Exploratory data analysis and regression modeling for the Kaggle "House Prices" dataset.

## Tools
- Python
- pandas, numpy
- scikit-learn
- RandomForest
- LightGBM

## What was done
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature preparation
- Training and evaluation of regression models
